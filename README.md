# 🛍️ E-commerce Lippan Art

A full-stack e-commerce web application built using **PHP and MySQL** for selling Lippan Art products.

## 🚀 Features

- User Authentication (Login/Register)
- Product Catalog & Collections
- Shopping Cart System
- Secure Checkout Process
- Order Tracking & Order History
- Product Reviews & Testimonials
- Admin Dashboard (Manage products, users, orders)
- B2B Order System
- Email Notifications (Order confirmation & updates)

## 🛠️ Tech Stack

- PHP
- MySQL
- HTML, CSS, JavaScript
- PHPMailer (for email system)

## 📦 Modules

- Customer Panel
- Admin Panel
- Order Management System
- Inventory & Category Management
